opensoma CLI는 SWMaestro 마이페이지의 모든 기능을 터미널에서 사용할 수 있게 해줍니다. 멘토링 생성, 회의실 예약, 공지사항 조회 등을 스크립트나 AI 에이전트에서 자동화할 때 유용합니다.

모든 출력은 JSON 형식입니다. `--pretty` 옵션을 추가하면 들여쓰기가 적용된 읽기 쉬운 형태로 출력됩니다.

```bash
opensoma dashboard show --pretty
```

명령어를 실행하기 전에 반드시 인증이 필요합니다. `opensoma auth login`으로 인증을 설정하세요.
